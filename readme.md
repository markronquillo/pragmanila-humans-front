Pragmanila Humans API Front End
--------------------------------

This project consumes the basic API _https://github.com/markronquillo/pragmanila-humans_.

This project is written using Vue.js framework.

To make this demo work. Kindly run the api using __php -S 0.0.0.0:8000__ and run this project using __php -S 0.0.0.0:8080__ or any port other than _8000_.
